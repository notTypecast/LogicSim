Simulation Tool

The simulation tool corresponds to the simulation mode. During simulation, other tools are disabled. The design view can be moved in the same way as when the move tool is active.

While in simulation mode, the simulation can be paused using the Simulation > Pause option. When paused, the simulation can be stepped by the simulation frequency (in ticks) using the Simulation > Step option. The simulation can then be resumed using the Simulation > Continue option. At any point during simulation, the Simulation > Reset option can be used to reset the simulation to tick 0. Finally, the Simulation > Stop option can be used to stop the simulation.

All of these options can also be found on the toolbar.
