View

The view menu allows for a few additional options. The toolbar can be toggled at View > Toolbar.

Wire color can be toggled at View > Wire Color. This is an option that applies to simulation mode. If toggled on, during simulation, wires that carry a logic 1 will turn green, wires that carry a logic 0 remain black and wires that carry HiZ will turn red.

There are also options to zoom in and out, as well as reset zoom level. Using these options from the View menu will result in the zoom happening around the center of the design area. Instead, CTRL + Scroll Wheel can be used to zoom in or out around the cursor.
