States

In digital circuits, probing a wire at a certain spot often results in two states, 0 or 1. In reality, there are certain voltage ranges that are assigned to each of these and, depending on the voltage of a wire, the corresponding state is decided.

Nevertheless, because of the complicated nature of real world circuits, it is not always possible to assign one of these to a spot on the circuit with certainty. This creates issues for digital design, where a simplified model is used.

3 State Model

LogicSim uses a 3 state model to deal with this issue. Any spot in a circuit can be either of:

  • Logic 0 (False)
  • Logic 1 (True)
  • HiZ (Unknown)

At the very beginning of a simulation, all outputs of non-input components are initialized to HiZ. Input components always output a 0 or 1, so their output is immediately set to that.

As the simulation progresses, components calculate their output based on their inputs and propagate it, often resulting in a valid circuit output.
