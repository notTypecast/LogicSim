Multiplexer

Multiplexers are control components that allow for selecting between multiple signals based on other select signals. Multiplexers are defined by the number of select signals they possess.

A multiplexer will have b select signals, 2b inputs and 1 output. Inputs are numbered 0 to 2b - 1. The select signals indicate a b bit binary number, which corresponds to the signal that will be propagated to the output.

LogicSim provides 3 multiplexers: a 2-1, a 4-1 and an 8-1 multiplexer. The number shown corresponds to the number of inputs, 2b.

Multiplexers also have an active low enable input. When this input is at 1, the multiplexer always outputs 0.
