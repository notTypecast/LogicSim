Decoder

Decoders are control components that allow for an input binary number to be mapped to a corresponding output. Decoders are defined by the number of inputs they possess.

A decoder will have b inputs and 2b outputs. Inputs are numbered 0 to b - 1, while outputs are 0 to 2^b - 1. The inputs indicate a b bit binary number that indicates which output will be 1. All other outputs will be 0.

LogicSim provides 3 decoders: an 1-2, a 2-4 and a 3-8 decoder. The numbers correspond to the numbers of inputs and outputs for the decoder.

Decoders also have an active low enable input. When this input is 1, the decoderâ€™s outputs will always all be 0.

If the enable signal is not connected (HiZ), all outputs will also be HiZ. If any of the inputs are not connected (HiZ), all outputs will also be HiZ.

A decoder with an enable signal can also be seen as a DeMultiplexer. If we consider the enable signal to be the input and the b inputs to be select signals, then we can say that the select signals decide which output the enable signal will be propagated to (in this case inverted, since enable is active low).
