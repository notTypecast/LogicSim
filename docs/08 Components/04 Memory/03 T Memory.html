T Memory Component

T memory components are memory components that store 1 bit of information.

These components have 1 main input, T, as well as a clock input. This input is used to toggle the state of the component.

Truth Table

T Q Q’
0 Q Q’
1 Q’ Q

Delay: 5 ticks

T components can either be latches or flip flops. Latches will only change their state based on their T input when the clock signal is at 1, whereas flip-flops will only do so during a positive edge (clock signal switch from 0 to 1).

Note that, if the state of the component is HiZ, toggling the component will have no effect on it, since toggling HiZ results in HiZ. These components must always be set first using the preset/clear inputs.
